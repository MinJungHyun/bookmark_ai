# image 작업 팁

- 첫 이미지는 미드저니로 생성하고 맘에드는 초안을 생성한다.
- 생성한 이미지로 다양한 포즈나 모습을 만들기위해 gemini 2.5 flash image를 이용한다
- 상품 합성하고 설명하는 영상을 만들고 싶다면 topview로 합성이 쉽게 된다
- 미드저니는 일관성이 부족하여 계속 사용하기는 어렵고 만들어낸 첫프레임을 가지고 영상을 만드는데 유용하다
